## Best Practices for Users

1. **When Offline**:
   - Continue working normally
   - Changes are queued automatically
   - Save when connection returns

2. **On Save Error**:
   - Check error message for guidance
   - Click Retry button if applicable
   - Check network connection
   - Sign in again if needed

3. **Connection Restored**:
   - Queued operations process automatically
   - Green notification confirms connection
   - No action needed from user
